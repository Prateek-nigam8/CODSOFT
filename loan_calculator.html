User Instruction

• Enter the loan amount you wish to borrow in INR (Indian Rupees) in the "Loan Amount" field.

• Set the interest rate percentage using the range slider or by manually entering the value in the "Interest Rate" field.

• Set the loan term in years using the range slider or by manually entering the value in the "Loan Term" field.

• Select the payment frequency from the dropdown list provided.

• Click on the "Calculate" button to see the loan summary.

• The loan summary will display the payment frequency, number of payments, loan amount, and monthly payment details.

Note: The user can adjust the loan amount, interest rate, loan term, and payment frequency as many times as they want before clicking the "Calculate" button to get the updated loan summary.
